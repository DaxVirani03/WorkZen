# Admin Dashboard - Quick Reference Guide

## 🎯 Overview
Complete admin dashboard with full CRUD operations and user/role management capabilities.

## 🔐 Access
- **URL**: `/dashboard` (after login)
- **Required Role**: Admin
- **Authentication**: Token-based (stored in localStorage)

## 📊 Dashboard Sections

### 1. Employees (Default View)
**Features:**
- View all employees in card grid
- Search by name, position, or email
- Filter by department
- Real-time status indicators (Present/Leave/Absent)
- Click card to view detailed employee panel

**Statistics:**
- Total Employees
- Present Today (green)
- On Leave Today (blue)
- Absent Today (yellow)
- Monthly Payroll Cost

**Actions:**
- Click employee card → View details panel
- Search bar → Filter employees
- Department dropdown → Filter by department

---

### 6. Settings (⚙️ USER & ROLE MANAGEMENT)
**Features:**
- Complete user management
- Role assignment
- User status control
- User CRUD operations

#### User Management Table:
**Columns:**
- User (Avatar + Name + ID)
- Email
- Role (Dropdown for inline editing)
- Department
- Status (Active/Inactive - clickable toggle)
- Actions (Edit/Delete)

#### Actions:
1. **Add User** (Top right button)
   - Opens modal form
   - Required: First Name, Last Name, Email, Password, Role, Company
   - Optional: Department
   - Roles: Admin, HR Officer, Payroll Officer, Employee

2. **Edit User** (✏️ Edit icon)
   - Opens modal with pre-filled data
   - Can update all fields except password
   - Click "Update User" to save

3. **Delete User** (🗑️ Delete icon)
   - Shows confirmation dialog
   - Permanently removes user

4. **Change Role** (Dropdown in table)
   - Inline role change
   - Auto-saves on selection

5. **Toggle Status** (Active/Inactive badge)
   - Click to toggle user active status
   - Green = Active, Red = Inactive

#### Role Descriptions:
Displays capabilities for each role:
- **Admin** 🛡️: Full system access, manage all users, CRUD all modules
- **HR Officer** 👥: Manage employees, approve leaves, track attendance
- **Payroll Officer** 💵: Process payroll, manage salaries, generate payslips
- **Employee** 👤: View own records, request leave, mark attendance

## 🔑 Keyboard Shortcuts & Tips

### Quick Navigation:
- Click sidebar items to switch views
- Use search bar to filter employees
- Click employee cards for quick details

### User Management Tips:
1. Go to Settings tab
2. Click "Add User" for new user
3. Use inline role dropdown for quick role changes
4. Click status badge to activate/deactivate users
5. Use edit icon for full user update
6. Delete only when necessary (shows confirmation)

### Leave Approval Tips:
1. Go to Time Off tab
2. Pending requests shown first
3. Click ✅ to approve instantly
4. Click ❌ to reject (will ask for reason)
5. Use status filter to view history

## 🚀 Quick Start Workflow

### As Admin - First Time:
1. Login with admin credentials
2. Navigate to Settings
3. Add new users with "Add User" button
4. Assign appropriate roles
5. Navigate to other sections to manage data

### Daily Tasks:
1. Check dashboard statistics
2. Review pending leave requests (Time Off tab)
3. Approve/reject leaves
4. Monitor attendance (Attendance tab)
5. Process payroll when needed (Payroll tab)
6. Generate reports (Reports tab)
7. Manage users as needed (Settings tab)
